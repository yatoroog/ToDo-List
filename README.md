# ToDo List CLI üìù

Small interactive console app to add, view, complete, and delete tasks. Data lives in memory for the current run.

## Quick start ‚ö°Ô∏è
- Prerequisite: Go `1.25.6` (or compatible).
- Run: `go run .`
- Type `help` in the prompt to see available commands.

## Commands
- `add` ‚Äî add a task (title, description, deadline/free text).
- `get` ‚Äî show one task by ID.
- `getAll` ‚Äî list all tasks.
- `del` ‚Äî delete a task by ID.
- `done` ‚Äî mark a task as completed.
- `help` ‚Äî show command list.
- `exit` / `quit` ‚Äî leave the app.

## Project layout
- `main.go` ‚Äî wires CLI to the task manager.
- `scanner/` ‚Äî user interaction: prompts, parsing, printing, error handling.
- `tasks/` ‚Äî task model and in-memory store.

## Notes
- Tasks are stored in `tasks.json` (created next to the binary) in JSON format.
- Restarting the app keeps existing tasks; delete `tasks.json` to reset.
- IDs auto-increment from 1.
